# ReactQRcodeReaderApp

ReactQRcodeReaderApp

## 動かし方

```bash
cd qrcode-reader && npm i && npm run start
```

## QR リーダーのインストールの仕方

```bash
npm i react-qr-reader --legacy-peer-deps
```
